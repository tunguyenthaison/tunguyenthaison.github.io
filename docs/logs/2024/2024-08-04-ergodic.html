Introduction

Basic notions

In this post I attempt to present a basic overview of ergodic theory (based on 3).

Let $(\Omega, \mathcal{M}, P)$ be a probability space. A map $T:\Omega\to \Omega$ (measurable transformation) is said to preserve the measure $\mu$ if

$$ \mu(T^{-1}(A)) = \mu(A) \qquad\text{for all}\;A\in \mathcal{M}. $$

  • ${\displaystyle \mu (T^{-1}(A))}$ has the form of a pushforward, whereas
  • ${\displaystyle \mu (T(A))}$ is generically called a pullback.

Almost all properties and behaviors of dynamical systems are defined in terms of the pushforward.

  • If $T:\Omega\to \Omega$ is measure-preserving, then an event $A\in \mathcal{M}$ is called invariant if $T^{-1}(A) = A$.
  • The collection $\mathcal{F}$ of all invariant events is the invariant $\sigma$-algebra.
  • If every invariant set $A$ either has measure $0$ or $1$, then $T$ is said to be ergodic.

Birkhoff's ergodic Theorem

Consider a flow $T_t:X\to X$ which preserves the probability measure $\mu$ on the space $(\Omega, \mathcal{M}, \mu)$.

Question
Given a function $\varphi:\Omega \to \R$, under what conditions will the time average of the function $\varphi$ over the trajectory $T_t(x^*)$ with initial condition $x^*$, $$ \langle\varphi \rangle _T(x^*) = \frac{1}{T} \int_0^T \varphi(T_t(x^*))\; dt $$ converge to the space average as $T\to \infty$ $$ \langle \varphi \rangle _\Omega = \int_\Omega \varphi \;d\mu . $$
By taking $\varphi$ to be the characteristic function $\chi_A$ of a set $A$ in $\Omega$ which is invariant under the flow $T_t$, one immediately shows that a necessary condition for answering this question in the affirmative is that all subsets invariant under the flow have measure $0$ or $1$. Flows with this property are said to be ergodic. Birkhoff's ergodic theorem says that this property is also sufficient.
Birkhoff's Theorem
For any function $\varphi \in L^1(\Omega, \mu)$, and for almost all $x\in \Omega$, we have $$ \lim_{T\to \infty} \frac{1}{T} \int_0^T \varphi(T_t(x))\;dt = M(\varphi) $$ for some function

The discrete setting

Let $\phi:\Omega\to \R$ be continuous. For $N\in \mathbb{N}$, we define

$$ \phi^{(N)} (x) = \phi(x) + \phi(Tx) + \ldots + \phi(T^{N-1}x) $$

as the correspondong Birkhoff sum. Birkhoff’s ergodic theorem in this setting asserts that uniformly

$$ \frac{1}{N}\phi^{(N)}(\cdot) \to \int_\Omega \phi\; d\mu \;\text{uniformly as function} $$

when $N\to \infty$.
